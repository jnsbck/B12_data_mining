#### Just a webscraper to track occupancy in my local climbing gym and a few methods to visualise it.

## Requires 
- python =< 3.5
- numpy
- pandas
- matplotlib
- urllib3
- bs4
- requests

## Use
B12_webscraping_tools.py contains a bunch of useful code for scraping and visualising data from
